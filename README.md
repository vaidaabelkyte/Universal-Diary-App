<h3><i>Project Overview</h3>
<p>
DiaryApp is a simple note manager application (personal blog). User can add notes to his/hers timeline, edit, update and delete them. <br>
Timeline (list of notes) is arranged by Date and Time.<br>
Project idea was to implement knowledge of MVVM design pattern and UWP data binding.
</p>
